i-SlideIn will connect to islidein.com server to display your i-SlideIn form.

== Description ==

i-SlideIn will connect to <a href="http://islidein.com" title="islidein.com">islidein.com</a> server to display your i-SlideIn form using your Account ID and Key fromt islidein.com.

**Follow this step to make it work**

After you register on islidein.com and create form you will get Account ID and Key.

Install plugin on your WordPrss site

Fill your Account ID and Key

It's done.

**i-SlideIn used JavaScript to get your form from <a href="http://islidein.com" title="islidein.com">islidein.com</a>**
